
### RSML Viewer WebApp

Visualize and analyse root systems stored in [RSML](http://rootsystemml.github.io/) format, as part of [CPlantBox](https://github.com/Plant-Root-Soil-Interactions-Modelling/CPlantBox).

**Features:**
- Upload `.rsml` or `.xml` root system files
- Inspect file metadata, properties, and functions
- Plot depth profiles (length / surface / volume per cm layer)
- Plot root system development over time
- Compute and visualise surface uptake fraction (SUF) for several hydraulic scenarios
- Compute and visualise root system conductivity Krs over time
- Edit: attach an artificial shoot segment; interpolate missing creation times
- Download the (modified) root system as `.rsml` or `.vtp`
