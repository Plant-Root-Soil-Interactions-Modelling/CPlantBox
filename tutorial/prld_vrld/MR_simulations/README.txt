Simulations are run with 

python3 multiruns_mays_wheat 'plant' 'MR Image size' 'root diameter' 'plant density'

thereby: 
'plant'... maize or wheat 
'MR Image size'...small or large 
'root diameter'...5.7 or 6.4 or 7.0
'plant density'... base or alt


the file Jobs.py creates bash files for all possible simulation Scenarios and starts them (for the use on a cluster)
